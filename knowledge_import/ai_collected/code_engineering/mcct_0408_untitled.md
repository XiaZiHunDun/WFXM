### Turn 10 (assistant)
The script is at `json_to_html_table.py`. It accepts a JSON array of objects and produces a styled HTML table.

**Usage:**

```bash
# Extract specific columns, write to file
python3 json_to_html_table.py data.json -c Name Age Email -o result.html

# All columns, print to stdout
python3 json_to_html_table.py data.json

# With a title
python3 json_to_html_table.py data.json -c Name Score -t "Report" -o report.html
```

**Options:**

| Flag | Purpose |
|------|---------|
| `-c`/`--columns` | Space-separated column names to extract (default: all) |
| `-o`/`--output` | Output file path (default: stdout) |
| `-t`/`--title` | Page and table caption title |

The HTML output includes inline CSS for readable formatting, properly escapes all values to prevent XSS, and handles `null` / missing fields gracefully.
