### Turn 12 (assistant)
All three strategies work. The script is at `xml_sampler.py` and supports:

- **`random`** — uniform random selection without replacement
- **`stratified`** — proportional sampling grouped by a tag (e.g. `department`), so each group is represented fairly
- **`systematic`** — picks every k-th element from a random starting offset

Usage:

```
python xml_sampler.py sample.xml -n 5 -s random --seed 42
python xml_sampler.py sample.xml -n 5 -s stratified --stratum-tag department
python xml_sampler.py sample.xml -n 3 -s systematic -o output.xml
```

The `--seed` flag makes results reproducible, and `-o` writes to a file instead of stdout.
