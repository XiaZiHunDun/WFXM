- **Basic correctness** (11 tests) — single element, two same/different, odd/even length, all same, majority at start/end/scattered
- **Edge cases** (7 tests) — empty list, bare majority (exactly n/2 + 1), exactly half (no majority), negatives, zero, large values (10^18)
- **Large inputs** (3 tests) — 10K and 50K element arrays with/without majority
- **Brute-force agreement** (12 parametrized cases) — verifies Boyer-Moore matches the Counter-based reference on varied inputs
