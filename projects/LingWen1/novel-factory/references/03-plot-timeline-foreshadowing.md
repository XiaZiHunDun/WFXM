# 《星陨纪元》剧情时间轴与伏笔

> 版本：参考书 v1.0 · 2026-06-20

## 全书结构速览

- **总章数**：360（v3.0 已发布）
- **三卷**：废土星火 / 星际争锋 / 宇宙守护
- **叙事宏循环**：起（卷1）→ 承（卷2）→ 转（卷3）→ 合（星光尾声）

## 三卷反派递进（写作钩子）

1. **卷1 暗皇**：惨败 + 「值得吗」
2. **卷2 虚空之王**：理解「完整」渴望，非单纯击杀
3. **卷3 虚无之主 / 本源之母**：拥抱式融合 + 说服重启冲动

林夜任务链：**不是打败，而是调和阴阳。**
