# 《星陨纪元》世界观与力量体系

> 版本：参考书 v1.0 · 2026-06-20  
> 对齐：全局大纲 v1.1 · 优化版大纲 v1.0 · 已发布 v3.0

## 一句话定位

废土科幻修真：从家园毁灭到星际联盟，再到宇宙级光暗调和；主题句——**守护不是征服，而是让生命安心存在**。

## 三卷地理与主题

| 卷 | 名称 | 章域 | 核心反派 | 主题 | 境界成长（叙事层） |
|----|------|------|---------|------|-------------------|
| 卷1 | 废土星火 | ch001–120（优化版至 ch131 含逃亡） | 暗皇 | 复仇→守护；惨败于暗皇 | 粒子→星火→脉冲→裂变 |
| 卷2 | 星际争锋 | ch121–260（优化版 ch132–270） | 虚空之王 | 消灭→理解 | 裂变→黑洞→创世 |
| 卷3 | 宇宙守护 | ch261–360 | 虚无之主→本源之母 | 光暗融合共生 | 创世→永恒 |

## 境界体系（两套术语对照）

叙事正文常用**粒子境链**；优化版增加**凡人界 / 星辰界 / 宇宙界**三级对照，修订时必须统一。

### 粒子境链（全书叙事节点）

每境分初、中、后、巅峰四段，共 28 个可用叙事节点。

| 境界 | 描述 | 主要出现 |
|------|------|---------|
| 粒子境 | 入门，操控微观粒子 | 卷1 初、中期 |
| 星火境 | 点燃星火，能量外放 | 卷1 中期 |
| 脉冲境 | 脉冲强化，战斗型 | 卷1 中、后期 |
| 裂变境 | 能量裂变，大范围攻击 | 卷1 末、卷2 初 |
| 黑洞境 | 引力、空间法则 | 卷2 主境界（星辰炼狱） |
| 创世境 | 创造法则，宇宙同源 | 卷2 末、卷3 |
| 永恒境 | 超越法则，与宇宙共存 | 卷3 终极 |

### 三级界对照（优化版 SSOT）

```
凡人界：炼气 → 筑基 → 金丹 → 元婴 → 化神
星辰界：星徒 → 星士 → 星师 → 星宗 → 星皇
宇宙界：域主 → 界王 → 永恒
```

| 粒子境链 | 三级界 | 分界说明 |
|----------|--------|---------|
| 粒子境 | 炼气/筑基 | 废土期入门 |
| 星火境 | 金丹 | 觉醒标志 |
| 脉冲境 | 元婴 | 战力成型 |
| 裂变境 | 化神（凡界巅峰） | ch091–100 主要突破；ch131 后入星辰界 |
| 黑洞境 | 星士~星师 | 星辰炼狱修炼 |
| 创世境 | 星宗~星皇 | 对标暗皇级 |
| 永恒境 | 域主~界王~永恒 | 最终决战 |

**关键分界**：卷1 ch120 逃亡结束踏入星辰界后，境界标定重置为**星徒**初、中期。

## 宇宙观与反派演化链

```
本源之母（光明面）
    │ 大灾变分离
    ├─ 虚无之主（阴影面）
    │       └─ 虚空之王（碎片 + 暗皇残留）
    └─ 暗皇（堕落守护者，「重启」执念）
```

| 存在 | 角色 |
|------|------|
| 本源之母 | 孤独创造者；结局需被说服「生命值得守护」 |
| 虚无之主 | 阴影面；林夜以拥抱式融合而非消灭 |
| 虚空之王 | 渴望完整；消灭它等于消灭宇宙另一半 |
| 暗皇 | 卷1 离去钩子；名台词：「如果你真的能守护到最后……替我问一句：这一切，值得吗？」 |

**结局回答**：「不值得。但不需要值得。它只需要存在。」

## 废土表层设定（卷1 开篇）

- **陨星 / 灵能污染区**：资源匮乏、变异生态
- **觉醒兽**：暗域驱使的非人威胁（ch001 父母之死）
- **暗域追杀**：林夜父母之死直接动因
- 地理扩张路径：废土 → 雷城 → 星际

## 写作时力量描写原则

1. 同一场景不混用未对照的两套境界名。
2. 卷2 起明确是否已「入星辰界」再标境界。
3. 暗皇 / 虚空之王 / 虚无之主力量描写应体现「法则之外」，但勿随意引入修真、暗能等未登记术语堆叠（参见 ch103 审核口径）。
