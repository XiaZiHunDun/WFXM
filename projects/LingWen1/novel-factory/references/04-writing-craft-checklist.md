## 开写前必读（顺序）

1. `workflow_state.json` — 当前维护态 / 批次
2. 本章大纲（若有）或 `03-plot-timeline-foreshadowing.md` 阶段定位
3. `05-consistency-rules.md` — 若修订旧章
4. 相关人物 / 境界条目（本目录 01–02）

## 本书文风锚点

| 维度 | 要求 |
|------|------|
| 开篇质感 | 废土黄昏、感官具体（气味、温度、触感）；少空洞形容词 |
| 情绪 | 父母之死、星月牺牲、裂痕和解等节点要有**身体反应**与沉默留白 |
| 对话 | 林夜前期稚嫩→沉稳；苏琳克制；铁蛋朴直；莫言锋锐后柔和 |
| 战斗 | 脉冲/裂变强调能量形态；黑洞境起加空间感知；避免纯数值堆砌 |
| 主题句 | 守护 > 复仇；理解 > 消灭；存在 > 值得 |

## 卷别语气

## 反 AI 味（本书高频）

参照 `webnovel-write/references/anti-ai-guide.md`，额外注意：

- 禁止「心中涌起复杂的情绪」类空泛总结
- 禁止连续三句以上排比抒情
- 预言能力用**行为后果**呈现，少「她预知到了未来」旁白
- 暗皇/本源级存在少用现代口语梗
- 章末钩子用**具体画面或半句对话**，非「命运的齿轮」

## 审查前自检（10 项）

- [ ] 境界名与所在界（凡人/星辰/宇宙）一致
- [ ] 小九形态与场景匹配
- [ ] 星月代词为「她」
- [ ] 铁蛋在 ch091–100 非死亡
- [ ] 苏琳 ch112 非写死
- [ ] 伏笔未提前剧透回收章内容
- [ ] 与全局大纲反派链不矛盾
- [ ] 无实验稿 `experimental/` 设定渗入正史
- [ ] 章号与文件名一致（若改仓库内章文件）
- [ ] 情绪曲线是否符合阶段表（见 03 文档）
