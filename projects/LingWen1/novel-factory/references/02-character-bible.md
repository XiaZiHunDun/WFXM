# 《星陨纪元》人物设定手册

> 版本：参考书 v1.0 · 2026-06-20  
> 原则：林夜 + 苏琳双主角贯穿；主角唯一性，不新增同级第二男主线。

## 核心四人组

### 林夜（男，主角）

| 项 | 内容 |
|----|------|
| 起点 | 废土少年；父母死于暗域追杀（ch001） |
| 弧线 | 复仇 → 守护；从惨败到理解光暗一体 |
| 境界 | 粒子境 → … → 永恒境 |
| 结局 | 与苏琳化为星光，永恒守护宇宙 |
| 关键台词关系 | 回应暗皇「值得吗」——存在本身即意义 |

**性格锚点**：动手前先想「值得保护的人」；后期倾向理解而非消灭。

### 苏琳（女，女主）

| 项 | 内容 |
|----|------|
| 能力 | 预言之眼；看见林夜多条未来含黑暗之路 |
| 角色 | 林夜的锚点；照见「唯一可能的共存之路」 |
| 核心冲突 | 预知堕落未来但**选择隐瞒** → 决策犹豫 → 裂痕 → 和解 |
| 生死 | ch112 为「还有一口气」昏迷，**非死亡**，为后续留空间 |
| 结局 | 与林夜同为星光 |

**描写要点**：预言代价体现为迟疑与判断偏差，避免旁白直接剧透「她看见了黑暗未来」。

### 铁蛋（男，守护者）

| 项 | 内容 |
|----|------|
| 出身 | 玄机阁 / 雷城机械师一脉 |
| 境界 | 黑洞境 |
| 伴侣 | 莫言 |
| 结局 | 凡人世界继续传承守护意志 |

**状态追踪（修订必查）**：

| 时间节点 | 状态 |
|----------|------|
| ch091–ch100 | 奄奄一息，被救（勿写死） |
| ch101–ch110 | 逐渐恢复 |
| ch111–ch120 | 清醒，参与战斗 |

### 莫言（女，剑修）

| 项 | 内容 |
|----|------|
| 出身 | 万剑宗弟子；宗门覆灭遗孤 |
| 弧线 | 追杀者 → 战友（卷1「羁绊」ch021–040） |
| 伴侣 | 铁蛋 |
| 名讳统一 | 万剑宗长老云清/周长老 → 统一 **莫天鸿** |

## 小九（灵兽线）

| 项 | 内容 |
|----|------|
| 身份 | 灵兽血脉传承者；大灾变见证者 |
| 出场 | 无记忆灵体跟随 |
| 结局 | 守护灵，自由游走星光 |

**形态日志（禁止声音/实体乱跳）**：

| 形态 | 特征 | 典型章节 |
|------|------|---------|
| 星灵形态 | 灵体、光环、可飘浮、穿墙、远距感知 | ch091–100, ch270–279, ch344–349 部分 |
| 人类形态 | 实体、可触碰、有尾巴、近战 | ch101–120, ch151–180, ch247–254 |

**切换规则**：星灵=能量体沟通；人类=实体交互与战斗。ch260 勿声音与实体无过渡跳跃。
