# 情感弧线检查清单

## 检查维度

### 1. 主角情感转折
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 转折是否有足够铺垫 | 至少2个关键事件触发 | P0 |
| 转折是否有人物内在逻辑 | 符合人物性格发展 | P1 |
| 读者能否提前预见 | 有伏笔/预兆 | P2 |

### 2. 女主情感弧线（苏琳示例）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| "隐瞒预言"是否有不可抗力因素 | 物理/法则层面的必要性 | P0 |
| 裂痕的产生是否有具体事件 | 导致什么后果，需明确 | P1 |
| 和解过程是否有足够心理描写 | 不能仓促/一句话带过 | P1 |

### 3. 婚后日常（如有十年空白）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 是否有关键情节点 | 至少3-5个重要事件 | P1 |
| 是否影响最终决战 | 为决战做铺垫 | P2 |

### 4. 情感锚点检查
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 相遇是否有画面感 | 具象化场景 | P1 |
| 牺牲是否有重量感 | 通过日常铺垫 | P1 |
| 结局是否有具象化描写 | 非概念性描述 | P2 |

### 5. 安静时刻检查（v2.0新增）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 每阶段是否有1-2个"安静时刻" | 非战斗/非推进的情感沉淀 | P1 |
| 安静时刻是否与主题相关 | 而非纯日常水字数 | P2 |
| 安静时刻是否能沉淀情感 | 为高潮做铺垫 | P1 |

**安静时刻类型：**
- 日常对话/内心独白
- 角色间的情感交流（非推进剧情）
- 环境描写（废土星空、废墟黄昏等）
- 回忆/闪回片段

**检查要点：**
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 中段（ch181-240）是否有安静时刻 | 防止节奏拖沓但情感空洞 | P1 |
| 最终决战前是否有情感沉淀 | 为"化星"结局做铺垫 | P1 |
| 安静时刻是否点缀而非主导 | 不能超过章节20% | P2 |

---

## 审核标准

### S级（通过）
所有检查项无P0问题，且P1问题≤2个

### A级（修复后复查）
存在P0问题但可修复，或P1问题≥3个

### B级（返回修改）
存在P0问题且修复难度大，或P1问题≥5个

### 不合格
存在≥2个P0问题
