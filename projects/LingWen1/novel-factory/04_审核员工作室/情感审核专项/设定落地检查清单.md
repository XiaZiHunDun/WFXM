# 设定落地检查清单

## 核心原则
核心设定必须有具体情节支撑，不能停留在"说明书层面"。

## 背景

v2.0之前的问题：
- 大灾变背景仅在框架层面提及，缺乏具体情节
- 五大奇点/星辰炼狱等设定感大于情节感
- 暗皇→虚空之王的能量转化缺少背景说明

---

## 检查维度

### 1. 核心设定情节落地
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 核心设定是否有具体情节支撑 | 而非仅在大纲中提及 | P0 |
| 大灾变背景是否有明确交代 | 具体原因/过程/后果 | P1 |
| 星辰炼狱修炼考验是否有具体内容 | 而非"进入炼狱"一句话 | P1 |
| 黑洞境修炼是否有具体考验 | 而非"修炼成功"结论 | P2 |

### 2. 设定与情节融合
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 设定是否服务于情节 | 而非情节服务于设定 | P0 |
| 设定是否有"代价"机制 | 使用能力需有代价 | P1 |
| 设定是否影响角色决策 | 而非仅作为背景 | P1 |

### 3. 重复设定差异化（v2.0新增）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 五大奇点两卷收集是否有不同本质 | 不能结构重复 | P0 |
| 第二卷奇点=力量/能量 | 需明确 | P1 |
| 第三卷奇点=记忆/情感/归还 | 需明确差异化 | P1 |
| 相同设定在不同卷是否有不同作用 | 而非简单重复 | P2 |

**推荐设计：**
- 第二卷收集五大奇点：获得力量突破创世境
- 第三卷收集五大奇点：归还/释放（而非获取），象征从"占有"到"守护"的转变

### 4. 伏笔回收节点标注（v2.0新增）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 代际守护伏笔是否有回收节点 | 铁蛋父亲→铁蛋→传承 | P1 |
| 小九记忆解锁是否有节点标注 | 星辰炼狱/奇点收集时 | P1 |
| 暗皇残留能量是否有明确来源 | 而非凭空出现 | P1 |
| 伏笔回收是否有情感落点 | 而非仅逻辑回收 | P2 |

### 5. 设定解释成本控制（v2.0新增）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 科幻+修真融合是否有一次性解释 | ch001-020建立认知 | P1 |
| 境界体系命名是否有内在逻辑 | 粒子→黑洞→创世→永恒递进 | P1 |
| 新设定首次出现是否有上下文 | 不能突兀引入 | P2 |

**推荐设计：**
- 在第一卷ch001-020通过铁蛋修机甲的细节，建立"此界法则即科学法则"的认知
- 为每个境界设计标志性战斗现象（黑洞境=引力透镜扭曲，创世境=法则几何纹路）

---

## 审核标准

### S级（通过）
- 所有核心设定有具体情节支撑
- 无P0设定空洞问题
- 重复设定有差异化处理

### A级（修复后复查）
- 存在设定空洞但有修复方案
- 重复设定差异可调整

### B级（返回修改）
- 存在≥2个P0设定空洞问题
- 设定沦为"说明书"而非情节驱动
