# 节奏健康度标准

## 核心原则
阶段划分需要变速，避免平均主义。

## 检查维度

### 1. 阶段类型判定
| 类型 | 特征 | 章数建议 |
|------|------|---------|
| 快节奏阶段 | 逃亡/决战/追击 | 15-25章，需高事件密度 |
| 慢节奏阶段 | 修炼/日常/情感 | 25-35章，需情感锚点 |
| 过渡阶段 | 衔接两段高潮 | 10-15章，快速推进 |

### 2. 阶段核心冲突验证
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 该阶段核心冲突是否明确 | 应能用一句话概括 | P0 |
| 记忆点事件是否存在 | 每个阶段至少1个 | P1 |
| 阶段之间过渡是否平滑 | 不能断裂/跳跃 | P1 |

### 3. 信息密度检查
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 第二卷（ch181-220）信息量是否足够 | 40章约10-15万字 | P0 |
| 是否存在"水字数"风险 | 核心事件不足 | P1 |
| 每20章是否有记忆点事件 | 防止节奏拖沓 | P1 |

### 4. 变速设计验证
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 快节奏阶段是否有足够事件 | 不能靠对话撑篇幅 | P0 |
| 慢节奏阶段是否有情感锚点 | 不能空洞无物 | P1 |
| 阶段长短是否服务于叙事 | 而非机械分配 | P2 |

### 5. 节奏密度检查（v2.0新增）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 每20章是否有"记忆点事件" | 至少1个强情节钩子 | P0 |
| 连续章节无推进是否超3章 | 防止读者流失 | P1 |
| 中段（ch181-240）是否有压缩空间 | 40章内应有高潮 | P1 |
| 日常/修炼段是否插入主线钩子 | 不能纯过渡 | P1 |

### 6. 中段膨胀风险检查（v2.0新增）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| ch181-240是否存在节奏拖沓 | 追击+婚礼+女儿跨度40章 | P0 |
| 生活场景是否打断主线紧凑感 | 应有机结合 | P1 |
| 每10章是否有情节推进 | 防止状态描写空转 | P1 |

---

## 审核标准

### S级（通过）
- 阶段类型判定合理
- 核心冲突明确
- 无P0信息密度问题

### A级（修复后复查）
- 存在信息密度问题但可修复
- 阶段类型判定有偏差

### B级（返回修改）
- 核心冲突不明确
- 存在≥2个P0信息密度问题
