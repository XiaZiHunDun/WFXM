# 配角价值验证标准

## 核心原则
配角在战力脱节后，仍需有不可替代的价值。

## 检查维度

### 1. 非战力价值定位
| 角色 | 可能的不可替代价值 | 验证问题 |
|------|------------------|---------|
| 铁蛋 | 凡人世界的精神领袖/机甲情感象征 | "机甲即枷锁"主题是否落地？ |
| 莫言 | 剑修意志传承/与铁蛋的情感锚点 | 背叛动机是否清晰？ |
| 小九 | 历史见证者/宇宙记录者 | 是否有主动选择成为守护灵的情节？ |

### 2. 独立支线剧情检查
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 是否有独立于主角的困境 | 需有自主选择 | P1 |
| 是否在结局有具象化呈现 | 非标签化描述 | P1 |

### 3. 道具情感承载检查
| 角色 | 道具 | 需验证的情感符号 |
|------|------|-----------------|
| 铁蛋 | 纳米机甲 | 是否承载父亲遗志/执念完成？ |
| 林夜 | （其他道具） | 是否在结局有呼应？ |
| 莫言 | 万剑宗传承信物 | 是否有独立道具线？ |

### 4. 角色结局分量检查（v2.0新增）
| 角色 | 需有明确的结局交代 | 问题级别 |
|------|------------------|---------|
| 林夜/苏琳 | 结局状态+告别场景+情感锚点 | P0 |
| 女儿 | 结局交代（存活/特质继承/告别场景） | P1 |
| 铁蛋 | 机甲崩解前的内心挣扎 | P1 |
| 莫言 | 背叛宗门的心路历程+独立支线 | P2 |
| 小九 | 记忆解锁节点+见证内容+守护灵选择 | P1 |
| 暗皇 | 结局明确（死亡/离开/融合） | P1 |

**检查要点：**
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 主角是否有告别场景 | 不能突然消失 | P0 |
| 关键配角结局是否有交代 | 女儿/铁蛋/莫言等 | P1 |
| 结局是否是"升华"而非"消散" | 强调he而非be | P1 |
| 凡人与神明的双线结局是否完整 | 烟火人间+星光永恒 | P2 |

### 5. 代际传承检查（v2.0新增）
| 检查项 | 标准 | 问题级别 |
|--------|------|---------|
| 代际守护伏笔是否有回收 | 铁蛋父亲→铁蛋→传承 | P1 |
| 传承是否有具象化载体 | 雕像/信物/学院/故事 | P1 |
| 女儿是否继承某种特质 | 而非仅"活着" | P2 |

---

## 审核标准

### S级（通过）
- 非战力价值定位清晰
- 无独立支线但有情感锚点
- 道具承载情感符号明确

### A级（修复后复查）
- 非战力价值定位模糊但可修复
- 存在独立支线但不完整

### B级（返回修改）
- 配角沦为纯啦啦队（无不可替代价值）
- 道具与情感无关联

```markdown
### 道具情感承载

```
