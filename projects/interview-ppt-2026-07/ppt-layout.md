# PPT 17 页 Layout 草图 · WFXM Butler

> **用途**：从 25 页候选素材压缩到 17 页关键 PPT。每页含 layout 类型 / ASCII 草图 / 内容填充 / 图引用 / 口播提示 / 时长。
> **总时长**：17 页 × ~60-90s/页 = 17-25 分钟（典型面试 PPT 演示窗口 15-25 分钟）。
> **设计原则**：每页只讲 1 个核心点；标题占 1/6，内容占 3/6，视觉占 1/6，speaker note 占 1/6。

---

## 🎨 全局设计约定

| 项 | 规格 |
|----|------|
| 画幅 | 16:9（1920×1080） |
| 字体 | 中文 思源黑体 / 英文 Inter（无衬线） |
| 主色 | #1E40AF（蓝）/ #DC2626（红，警示用）/ #16A34A（绿，通过） |
| 辅助色 | 5 色编码（接入黄 / 调度蓝 / 能力绿 / 子代理紫 / 外部灰） |
| 字号 | 标题 36pt / 正文 18pt / 注释 12pt |
| 数字锚点 | 字号放大到 48-60pt，必显眼 |

## ⚠ 制作 PPT 时的 5 条纪律

1. **每页只讲 1 个核心点**——贪多必失
2. **数字放大到 48-60pt**——1254 / 7 / 15 / 0.8 要显眼
3. **不要把 mermaid 源码当 PPT 图**——必须用 `png/` 目录里的 PNG
4. **speaker note 用 `talking-points/` 里的口播稿**——别自己写
5. **打印 cheat-sheet.md 带进面试**——手机备用
