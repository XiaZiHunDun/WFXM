# 04 · 白板画法指南

> **用途**：面试官说"画一下你的架构"时用。**白板画图比 PPT 更能展示工程深度**。
> **核心原则**：**30 秒画完、3 分钟讲完**，不要画细节。

## 画图心法（务必记牢）

| 心法 | 说明 |
|------|------|
| **30 秒画完** | 超过 30 秒说明画太细，停下来 |
| **3 分钟讲完** | 画完不讲等于没画 |
| **大框 + 关键词** | 不要写完整句子，关键词即可 |
| **数字锚点** | 1254 / 7 / 3 / 5 这种数字要显眼 |
| **失败案例** | 画完后讲"我之前在这里踩过坑……"——加分项 |
| **允许涂改** | 边画边改比画完美更真实 |
