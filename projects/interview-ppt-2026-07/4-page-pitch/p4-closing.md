## 五、视觉一致性提示（极重要）

> **P4 的灵魂标签字必须与 P1 完全相同**：
> - 字号相同（60-72pt）
> - 颜色相同（#1E40AF）
> - 位置相同（居中）
> - 字体相同

> 这是"视觉重复加深记忆"的核心技巧——面试官在 P1 看过一遍标签，到 P4 再看到一次，会形成强印象。

## 六、可能追问预案

| 追问 | 锚点 |
|------|------|
| LLM 没有规划，那 Agent 是怎么工作的？ | "LLM 做的是'下一步最佳动作'的模式匹配，Agent 框架提供规划脚手架——工具列表、记忆召回、子代理委派都是脚手架" |
| 工作记忆 + 长期记忆 + 元认知具体怎么工程化？ | "工作记忆 = 当前会话上下文；长期记忆 = MEMORY/Owner/项目 SSOT；元认知 = 漂移检测 + 自我压缩 + 失败回退" |
| 未来 Agent 会怎么发展？ | "短期靠工程化补足（记忆/工具/权限）；中期等 LLM 内置规划能力；长期是多模态 + 持续学习" |
| 那自建 Loop 还有意义吗？ | "短期有意义——框架自带能力是负债不是资产；中期看 LangGraph 是否补足微信 + 多项目 + 分层记忆" |
