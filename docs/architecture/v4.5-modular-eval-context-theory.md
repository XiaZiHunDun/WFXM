## 1. 七元组扩展

在 v4 五元组 \((\mathcal{G}, \mathcal{L}, \Pi, \mathcal{M}, \mathcal{A})\) 上增加：

| 符号 | 名称 | 工程锚点 |
|------|------|----------|
| \(\mathcal{E}\) | Eval 观测代数 | `butler/eval_integration/` · suites × sinks |
| \(\mathcal{T}\) | Context 变换代数 | `butler/core/context_transform_registry.py` |
| \(\mathcal{P}\) | Port 边界 | `butler/contracts/eval_ports.py` · `context_transform_ports.py` |

\[
\text{Butler}_{4.5} = (\mathcal{G}, \mathcal{L}, \Pi, \mathcal{M}, \mathcal{A}, \mathcal{E}, \mathcal{T}, \mathcal{P})
\]

**设计原则**：\(\mathcal{L}\)（Loop）内核不变；\(\mathcal{E}\)、\(\mathcal{T}\) 以 Adapter + Port 接入，禁止 OSS 直 import 穿透 `agent_loop`。

## 2. 扩展公理（A8–A11）

## 3. 形式化对象

### 3.1 Eval Suite

\[
\epsilon = (\text{id}, \text{layer}, \text{runner}, \text{sinks}, \text{schedule})
\]

- `layer` \(\in \{L\text{-}A, L\text{-}B, L\text{-}C, L\text{-}D\}\)（见 [`agent-testing-strategy-2026-06.md`](../plans/decisions/agent-testing-strategy-2026-06.md)）
- `sinks`：多后端并列 SSOT，不强一致双写

### 3.2 Score Sink

\[
\Psi_i = (\text{backend}, \text{namespace}, \text{schema\_version}, \text{write}, \text{read})
\]

首版 backend：`langfuse` · `junit` · `audit` · `deepeval`（opt-in）· `ragas`（opt-in）

### 3.3 Context Transform

\[
\tau_m: \mathcal{C} \times \Theta_m \rightarrow \mathcal{C}, \quad m = (\text{provider}, \text{model\_pattern})
\]

\(\mathcal{C}\) = API message 列表；\(\Theta_m\) = 可热更新参数（YAML profile + overrides）

## 4. 定理

### T-CTX-1（变换组合确定性，L1）

\[
\text{prepare}(c) = \tau_{m_k} \circ \cdots \circ \tau_{m_1} \circ \text{compress}(c)
\]

步骤全序由注册表 `priority` 决定；验证：`tests/test_context_transform_registry.py`

### T-CTX-2（热更新不破坏 Loop 不变量，L1）

`/model`、fallback 换 client 后，`refresh_model_binding()` 同步 `max_context_tokens` 与 transform profile；门控状态 \(\sigma\) 不变。验证：gateway sim + fallback 单测

### T-EVAL-2（多 Sink 弱一致，L2）

统一 `EvalReport v1` 索引各 sink 引用；各 sink 保留完整 payload。验证：`tests/eval_integration/test_report_schema.py`

### T-EVAL-3（反馈调参收敛，L4/有条件）

若 \(\text{score}(\epsilon) < \theta\) 且 trigger 允许，则 \(\Theta_m \leftarrow \Theta_m + \Delta\)，\(\|\Delta\| \leq \delta_{\max}\)。依赖 LLM 评分噪声；与 OT2/G1-04 同级诚实声明。
